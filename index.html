IT HAS TO BE EASIER THAN THIS

DECEMBER 2nd 2017

I am new to programming. Today I made my first attempt to use GitHub to build and host a simple website.

I tried the various WYSIWYG methods like Jekyll and could not make any progress with them.

I tried Github Pages and drew a blank.

I then switched to Atom editor and just typed the codes manually and then uploaded them to my standard GitHub repository. And this is the result.

I suspect that the terminal-driven short-cuts are only useful if you are already quite skilled. They are promoted for newbies but I find that actually as a total newbie I prefer the simple keyboard bashing process. Maybe the physical effort of typing helps me commit the process to memory?
